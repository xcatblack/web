Setting up Apache Hive is relatively simple provided Hadoop is installed.

We will unzip the source tarball into /opt/hive. All that is required next

ensure that .bashrc is set properly (both of those steps are completed if

the hadoop instructions were followed) and then initialize the database.

			cd $HIVE_HOME
			schematool -dbType derby -initSchema

And that's it! If schematool initialized correctly, Hive is up and running.

There is, of course, much more advanced configuration that could be done.

However, for the purposes of this tutorial, that is all we need.

Hive can be connected to using beeline:

			beeline -u jdbc:hive2://
